# Examples

This directory is reserved for instantiated project examples and documentation snapshots.

## Policy

- Schema sources live in `RE-schema/`.
- Reusable documentation scaffold lives in `docs/`.
- Example project content belongs here or in a downstream project repository.
- No authoritative schema files should be stored here.
